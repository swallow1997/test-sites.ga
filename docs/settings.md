## Получить ключ аффилиата

Управление настройками -> Настройки -> Настройки V3Project

## Агенты через крон

В настройках файла ``@common/config/main.php``

```php
'cmsAgent' => [
    'onHitsEnabled'     => false
],
```

## AssetManager - linkAssets

```php
'assetManager' =>
[
    'linkAssets'        => true,
],
```


## Отключить dev режим

Дополнительно -> Инструменты -> Информация -> prod/dev
